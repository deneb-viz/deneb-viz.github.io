# Resources and Examples

A curated list of resources (specifically) for using Deneb. If you have one to recommend (or would like to add your own), please [create an issue with the details](https://github.com/deneb-viz/deneb-viz.github.io/issues/new)!

## Websites and Template Repositories

Many of our community members have been creating and sharing their work, with many making them available as reusable templates, so you can get started and adapt quickly.

- [Kerry Kolosko](https://kerrykolosko.com/) - many Power BI-specific visualization articles, including worked
  examples for Deneb.
- [Thys | Deneb Templates](https://thysvdw.github.io/) - a growing site of Deneb templates, examples and how-to's.
- [Andrzej Leszkiewicz | Deneb and Vega in Power BI](https://powerofbi.org/dataviz-galaxy-power-bi-deneb-vega/) - Andrzej's overview of using Deneb w/Vega in Power BI, as well as [examples and JSON source](https://powerofbi.org/deneb-vega-data-visualization-examples/)
- [DataVogue](https://data-vogue.com/) - site of [Aleksandra Godlewska](https://www.linkedin.com/in/aleksandra-godlewska-b87032150/) and [Joanna Sękiewicz](https://www.linkedin.com/in/jsekiewicz/)'s work, with a number of Deneb examples and tutorials.
- [Shad Frugui | Vega-Lite](https://github.com/shadfrigui/vega-lite) - [Shad Frigui](https://twitter.com/shadfrigui)'s Vega & Vega-Lite examples, including some for Deneb.
- [PBI-David/Deneb-Showcase](https://github.com/PBI-David/Deneb-Showcase) - [Davide Bacci](https://www.linkedin.com/in/davbacci)'s Deneb showcase (many including sample workbooks).
- [PBIQueryous/Deneb](https://github.com/PBIQueryous/Deneb) - [Imran Haq](https://www.linkedin.com/in/imranhaqbi)'s GitHub repository, containing templates and specifications.
- [Giammaria/Vega-Lite-Techniques](https://github.com/Giammaria/Vega-Lite-Techniques) - [Madison Giammaria](https://www.linkedin.com/in/madison-giammaria-58463b33)'s GitHub repository of Vega-Lite techniques.
- [Giammaria/Vega-Visuals](https://github.com/Giammaria/Vega-Visuals) - [Madison Giammaria](https://www.linkedin.com/in/madison-giammaria-58463b33)'s GitHub repository of Vega visualizations, containing specifications accompanied by Power BI files that include Deneb implementations.
- [Flynnxx1/Deneb-Vega-Showcase](https://github.com/Flynnxx1/Deneb-Vega-Showcase) - [Ying Fu](https://www.linkedin.com/in/yingfu-nl)'s GitHub repository of Deneb and Vega-Lite showcase examples.
- [PowerBI-Tips/Deneb-Templates](https://github.com/PowerBI-tips/Deneb-Templates) - GitHub repository for Deneb templates, curated by PowerBI.tips. Contributions and pull requests are welcome.
- [Enterprise DNA Deneb Showcase](https://forum.enterprisedna.co/c/deneb-showcase) - section of the Enterprise DNA forums dedicated to demonstrate and share Deneb templates and questions.
